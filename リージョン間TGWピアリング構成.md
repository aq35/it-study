## リージョン間TGWピアリング構成（ポケモン版）

---

### 登場人物（拡張版）

| 役割 | ポケモンで例えると |
|------|------------------|
| リージョンA | **カントー地方** |
| リージョンB | **ジョウト地方** |
| TGW-A | **カントーの列車駅（タマムシ駅）** |
| TGW-B | **ジョウトの列車駅（コガネ駅）** |
| TGW間ピアリング | **地方間を結ぶ特急列車** |
| VPC-A | **カントーのポケモンセンター** |
| VPC-B | **ジョウトのポケモンセンター** |
| オンプレミス | **サファリゾーン（カントー側にのみ入口がある）** |
| DXゲートウェイ | **サファリゾーンの受付** |

---

### 構成図（ポケモン版）

```
サファリゾーン（オンプレ）
        ↓ そらをとぶ（Direct Connect）
   サファリ受付（DXゲートウェイ）
   「カントーとジョウト両方の部屋に案内できます」
   ← VPC-A + VPC-B 両方を許可プレフィックスに追加
        ↓
  タマムシ駅（TGW-A）  ──特急列車──  コガネ駅（TGW-B）
  ルートテーブルに              ルートテーブルに
  ジョウト行きを追加            カントー行きを追加
        ↓                              ↓
カントーのポケモンセンター      ジョウトのポケモンセンター
     （VPC-A）                       （VPC-B）
```

---

### なぜVPCピアリングではダメなのか

#### ❌ VPCピアリング＝「ヒトカゲとゼニガメが友達になる」だけ

```
サファリゾーン
     ↓
     ✋ 壁（入れない）

カントーのポケモンセンター ←友達→ ジョウトのポケモンセンター
       （VPC-A）    ピアリング      （VPC-B）
```

> VPCピアリングは「**2匹が仲良くなる**」だけ。  
> サファリゾーンのおじさんは「**友達の友達は知らん**」というルール（**中継不可**）を持っている。

---

### ポケモンで覚える3つの制約

| 制約 | ポケモンで言うと |
|------|----------------|
| ピアリングはトランジット不可 | 「友達の友達には会えない」おじさんルール |
| ルートテーブルに手動追加必要 | 駅員さんに「ジョウト行きの時刻表を貼って」と頼む必要がある |
| 許可プレフィックスの明示指定 | 受付に「カントーとジョウト両方の部屋番号」を申告する必要がある |

---

### 正解構成をひとことで

> **「サファリゾーンからジョウトに行くには、カントーの駅（TGW-A）経由の特急列車（TGWピアリング）に乗るしかない。友達経由（VPCピアリング）では行けない。」**
